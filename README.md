# Web Scrapers Collection

This repository contains a collection of my web‑scraping projects. Each folder below represents a different scraper with its own functionality.

For demo videos and usage, check my [YouTube channel](https://www.youtube.com/@SubhanAli-n7o).

---

## Scrapers Included

### 1. YouTube Scraper  
**Folder:** `youtube-scraper`  
Scrapes YouTube data such as video titles, links, and statistics.  
[Demo Video](https://www.youtube.com/watch?v=O8ur10GAEv4)

### 2. Binance Scraper  
**Folder:** `binance-scraper`  
Scrapes cryptocurrency data from Binance exchange.  
[Demo Video](https://www.youtube.com/watch?v=jodY4W_1rNc)

### 3. DEX Screener Signal Tracker  
**Folder:** `dexscreener-signal-tracker`  
Tracks signals from DEX Screener for trading opportunities.  
[Demo Video](https://www.youtube.com/watch?v=1L369m5L3AY&t=1s)

### 4. Instagram Lead Generation  
**Folder:** `instagram_LeadGen`  
Scrapes leads and contact information from Instagram public profiles.  
[Demo Video](https://www.youtube.com/watch?v=wLmeTd680g0)

### 5. Real‑Time Cookie Extractor  
**Folder:** `real-time-cookie-extractor`  
Extracts cookies in real‑time from browsers for automation purposes.  
[Demo Video](https://www.youtube.com/watch?v=BibA3hOS3m8)

### 6. TradingView Scraper  
**Folder:** `tradingview`  
Scrapes charts and trading signals from TradingView.  
[Demo Video](https://www.youtube.com/watch?v=MKZG7XTtqrU)

### 7. Google Maps Scraper  
**Folder:** `googlemap`  
Scrapes business data, addresses, and contact information from Google Maps.  
[Demo Video](https://youtu.be/2fBVfbRz1sY)
